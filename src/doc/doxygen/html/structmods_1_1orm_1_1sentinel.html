this should be called when you create a sentinel player for the first time

Handy little script to restore a sentinel's primary and secondary weapons: You can pass it space separated primary keys run ./sql/restore-sentinel.sh $USER_PK_ID [...$USER_PK_ID_N] Then run sql-copy, then sql/restore.sql Primary_choice will be either "MP5" or "SASG12"
