Detailed Description

When events are called **

Events are called after everything that's needed to be done for the event type. For example, the OBJECT_FROM_ROOM event is published when the object is fully removed from the room. It is not called before or during the object being extracted from the room. These events are to be emitted when the action the event represents is complete.
