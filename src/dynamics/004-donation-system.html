Donation system

  • An item may be donated to "the gods" so to speak, in return get perks

Example: donate sniper rifle, get money

  • User donates item to "the gods"
  • User receives money for item

Example: donate sniper rifle, add N percent to luck

  • User donates item to "the gods"
  • User receives +N% chance to find X-tier loot, where X is determined by player level

Example: donate sniper rifle, enable Divine Intervention Mod

Divine Intervention

  • User donates item to "the gods"
  • User attacks NPC
  • NPC rolls a crit
  • Roll User LUCK attribute (+N% * LUCK attribute)
  • User may or may not dodge crit (+N% helps chances)
