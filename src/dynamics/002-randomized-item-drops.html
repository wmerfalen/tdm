Randomized Item Drops

  • Items have a random chance of dropping

Example: Tiered drops

  1. Player kills NPC.
  2. 66% chance for uncommon item to be populated in NPC corpse
  3. 33% chance for uncommon item to be populated in NPC corpse
  4. 10% chance for rare item to be populated in NPC corpse
  5. 1% chance for legendary item to be populated in NPC corpse
  6. 0.1% chance for ultra legendary item to be populated in NPC corpse
  7. 0.01% chance for mythic legendary item to be populated in NPC corpse

Attribute modifiers

  • STR,INT,DEX,WIS,CHA affected
