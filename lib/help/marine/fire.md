# Launch a frag grenade

# Ability
This command is used after you have attached an underbarrel attachment to your assault rifle.

# Usage
`marine:fire <direction> <distance>`

# Step one: Attaching to an Assault Rifle
`marine:attach_m203`

# Step two: Using it
`marine:fire west 2`

# See also:
`marine:attach_m203`
`marine:detach_m203`
